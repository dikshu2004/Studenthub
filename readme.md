# StudentHub

StudentHub is a personal productivity platform designed to help students manage their academic life in one place. The idea behind this project is to build a website that grows along with my learning journey. Instead of creating multiple small projects, I am continuously improving StudentHub by adding new features and technologies as I learn them.

The project starts with HTML, CSS, and JavaScript to build a responsive and interactive user interface. As I progress, I plan to migrate it to React, enhance the design with modern UI practices, integrate APIs, and eventually develop a full-stack version using Node.js, Express.js, and MongoDB.

StudentHub will include features such as a study dashboard, notes, task management, timetable, Pomodoro timer, study resources, progress tracking, and other tools that make student life more organized and productive.

This project reflects my commitment to continuous learning, clean code, and building real-world applications while improving my frontend and full-stack development skills.
